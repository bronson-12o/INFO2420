Web Site Critiques

Trader Joe's - https://www.traderjoes.com/home

  1. First Impression: The Trader Joe's website has a clean, open, and inviting design. It makes a user want to explore their brand and can do so simply and quickly. It gives off the specific energy the store is going for to set them apart from other grocery stores.
  2. Information Flow: This website has a top navigation ribbon that leads to their specific pages for more details. This ribbon stays with you even when you scroll down the page. The ribbon displays "Products, Flyer, Discover, Recipes, and Listen." Having the information all hyperlinked with buttons or pictures leaves a good display for the user. As well, it is organized in a grid format making it easy for the user to explore.
  3. Structure: The site structure makes sense and is easy to navigate. The headings are a little different from usual websites, but again set the quirky store apart from others. The navigation guides the user throughout the site on the top and bottom with more resources.
  4. Graphics/Multimedia: The graphics and multimedia add to the site, creating a want to learn more as a user. The seamless font ties into pictures that coerce a user to delve deeper into one of the five main categories. The multimedia is integrated well and supports the flow and look of the site.
  5. Load Time: The load time is strong and loads quickly. All elements are there with no blanks or missing .jpeg files anywhere. The links all load quickly and to the correct area. This also adds to the user experience, saving them time and patience to enjoy the site.
  6. Text Readability: The text and coloring is good. It adds to the site's color scheme well. I wish some wording could be bigger in some announcements to make it seem more drawing to click on. Overall, it creates a smooth experience that ties into the design of the site.
  7. Navigation: Yes, the site does have a navigation bar that is simple and useful. Due to the lack of needing to display online items, a user can get to where they want in 2â€“3 clicks which is great! The navigation is simple and well-organized.
  8. Color Scheme: I think this is where this site excels with having an overarching color theme of red, blue, white, and gray. As well, they create little subsections that are their own theme to make announcements "POP!" and draw in a user from the usual theme. This theme carries through all links and windows as well.
  9. Responsive Design: The website responds well to different screen sizing and either makes art or pictures bigger or disappear to allow a reader to read the text instead of being blocked by pictures everywhere. It keeps design on computers and mobile devices as well. This allows any audience to use the website and have it be a user-friendly experience.
  10. Notable Feature: I would say it makes the user feel like a loyal customer using the site. As well, the "The Fearless Flyer" section of news and any new ideas for the company can be announced and read and enjoyed by a user. It pulls a user into something different than the usual announcements or sales on other websites.
